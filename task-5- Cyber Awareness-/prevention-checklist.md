# Stay Safe Online: Prevention Checklist for Indian College Students 

-  Verify before clicking: Hover over links to check real URLs (e.g., vitbhopal.ac.in, not .co.in). Use 
official apps/sites only. 
-  Enable 2FA: Activate two-factor authentication on email, Google Workspace, and banking apps. 
-  Spot red flags: Ignore urgent demands for personal info; poor grammar or unknown senders = 
scam. 
-  Use strong, unique passwords: Employ a password manager like Bitwarden; change defaults. 
-  UPI Safety 1: Set UPI PIN limits (e.g., ₹5,000/day) and enable transaction SMS alerts via 
BHIM/Google Pay. 
-  UPI Safety 2: Never share UPI PIN/MPIN; use virtual payment addresses (VPA) without revealing 
bank details. 
-  Scan devices: Install free antivirus (e.g., Avast) and run weekly scans. 
-  Privacy first: Lock social media profiles; avoid posting Aadhaar/PAN publicly. 
-  Backup data: Use Google Drive for academic files to counter ransomware. 
-  Report incidents: Immediately visit cybercrime.gov.in or call Helpline 1930. Save evidence 
(screenshots). 
Stay vigilant—prevention beats cure!
